# Corrode Browser 0.1 Beta
A premium, modern custom web browser built with Python + PyQt5 + QtWebEngine.

## FEATURES
- Multi-tab browsing
- Custom homepage
- Splash screen
- About dialog
- Modern theming
- Rust gear branding
- Back/Forward/Reload/Home buttons

## INSTALLATION
pip install PyQt5 PyQtWebEngine

## RUN
python browser/main.py
